2.ミーティングの概要

【作業内容】
開発項目と優先度、詳細機能について話し合った。
以下のような結果になった。
　　
開発項目 優先度 詳細機能
自律走行 A LiDAR,超音波を使用し自己位置推定をし自立走行をする
エレベータ B ロボットアーム、物体認識を使用しエレベータを乗り降りする
ロボットアーム A ペットボトル程度の重さの物を持て、物の受け渡しをするために使用する
受け渡し A 人を認識し、物の受け渡しをする
物体認識 A 受け渡しするの種類と位置を機械学習を用いて認識する
人間感知 A 人間を認識する
人間分類 B 人間を認識し、男女、年齢、身長を機械学習を用いて分類する
人間追尾 B 物の受け渡し安くするため、対象に接近する
表示案内 B 情報を画面に表示する
音声案内 C 情報を音声として発信する
写真撮影 C ロボットアームを使用し、観光地などの写真撮影を代行する
アプリ C ロボット本体に設置したディスプレイで、構内のマップや道案内をする
マニュアルモード B ラジコンを使用しマニュアル操作ができるようにする
